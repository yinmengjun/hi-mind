a)  通过SQL代码实现，查询出该表的所有记录（重复记录只显示一条）

| 6000121 | 9000000001 | 上海分行 | 100.00 | 20120701 | 20000 |

**Query 1**:

    SELECT
    	* 
    FROM
    	t03_acct_trade_detail a 
    WHERE
    	a.ROWID = (
    	SELECT
    		MIN( b.ROWID ) 
    	FROM
    		t03_acct_trade_detail b 
    	WHERE
    		b.cust_id = a.cust_id 
    		AND b.acct_no = a.acct_no 
    	)

**Query 2**:

    
    DELETE 
    FROM
    	t03_acct_trade_detail a 
    WHERE
    	a.ROWID > (
    	SELECT
    		MIN( b.ROWID ) 
    	FROM
    		t03_acct_trade_detail b 
    	WHERE
    		b.cust_id = a.cust_id 
    	AND b.acct_no = a.acct_no 
    	)
